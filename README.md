- REQUEST:

    - CLASS: ./request/REQUEST.py

    - How to run: ./main_request.py

    - How to config: ./request/config.py

- SERVICE: 

    - CLASS: ./service/SERVICE.py

    - How to run: ./main_service.py

    - How to config: ./service/config.py
